What is Blurple Is A Color?

Blurple Is A Color is my fourth of July break project for the summer class I am taking at City College of San Francisco. As per the project's assignment, I have created a website following a specified requirements, all of which can be found at the bottom of this page, or by clicking here.

Since the topic of my website was left open for me to choose, I have decided to dedicate my website to my love of the color blurple. Blurple Is A Color will showcase various themes, images, and wallpapers featuring shades of blurple hues, as well as pink, purple, and blue tones.

Clicking on "Blurple Samples" in the top navigation will take you to a page displaying these vaious themes, images, and wallpapers.

The following is a list of specific requirements for this project:

  • Appropriate DOCTYPE for HTML5
  • Opening and closing html tags properly enclosing the entire HTML document
    • Don't forget the lang attribute!
  • Opening and closing head tags, properly enclosing the entire head section
  • Opening and closing body tags, properly enclosing the entire body of the webpage
  • A character encoding declaration of UTF-8
  • A comment in the head section that states the purpose, author, and date of page creation
  • An appropriate title that appears in the title bar of the browser, properly placed in the head section of the HTML document between title tags
    • Each page should have a unique title.
  • An appropriate meta tag description based on the main topic of your web page
  • The body section must include the HTML5 header and footer tags with appropriate content in those sections
  • Use the following tags at least once on one or both webpages: h1, h2, p, br, hr, img, a, ol/ul/dl
    • Create at least one table with tabular data.
  • Include at least one internal link on both pages
    • The home page should link to the second page; the second should link to the home.
    • Include at least one external link on either/both pages.
  • Source in your CSS file to both of your webpages
  • Defined background color and/or image for your webpages
    • The image can be the same or different for each webpage
  • Defined styling for your ul/ol/dl
  • Set font styling of your text
  • Changed color of a single word on either page
  • HTML and CSS code valid by using the W3C validator that mentioned in the lecture
    • Click here to visit validator
  • Submitted zip file of the directory with all project files
